# 圆盘纪元 · 桌游说明书

## 游戏简介

四人阵营对战桌游。每队两人，玩家打出卡牌控制角色移动、攻击与释放技能，击破对方主堡即可获胜。

---

## 快速上手

### 1. 开始游戏
- 打开 `index.html` → 点击「开始游戏」
- 系统自动发 3 张起始手牌

### 2. 回合流程
每个回合自动推进：

| 阶段 | 说明 |
|------|------|
| 抽牌 | 能量+1（上限10），抽1张牌 |
| 行动 | 60秒内出牌、操作 |
| 结算 | 执行DOT伤害、护盾消退 |
| 结束 | 检查胜负，切换回合 |

---

## 操作指南

- **出牌**：点击手牌 → 点击己方半场空格部署
- **移动**：点击己方单位 → 点击可移动空格
- **攻击**：点击己方单位 → 点击敌方单位或主堡
- **技能**：点击己方单位 → 弹出技能选项
- **结束行动**：点击「结束行动」按钮，或等待60秒倒计时结束

---

## 四大阵营

| 阵营 | 特点 |
|------|------|
| 🔵 寒霜 | 冰系控制，减速冻结 |
| 🔴 赤焰 | 火焰爆发，高伤秒杀 |
| 🟢 森木 | 持续恢复，消耗战强 |
| 🟡 厚土 | 防御护盾，持久攻坚 |

---

## 胜负条件

- **胜利**：对方主堡 HP 归零
- **失败**：己方主堡 HP 归零
- 可随时点击「投降」认输

---

## 常见问题

**Q: 能量不够怎么办？**
每回合自动+1，合理规划出牌节奏。

**Q: 如何查看角色技能？**
点击己方角色，技能选项会自动弹出。

**Q: 手牌太多怎么办？**
手牌上限10张，超出时自动弃牌。
